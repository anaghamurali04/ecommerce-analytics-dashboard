# E-Commerce Analytics Dashboard

An end-to-end E-Commerce Analytics Platform built using Python, PostgreSQL, React, Recharts, and Power BI.

The platform analyzes e-commerce business data and provides interactive dashboards for revenue tracking, customer review analysis, seller performance monitoring, and business KPI reporting.

---

# Project Overview

This project was developed to transform raw e-commerce data into meaningful business insights through data processing, analytics, and visualization.

The project follows a complete analytics workflow:

1. Data Collection
2. Data Cleaning and Processing
3. KPI Generation
4. Database Storage
5. Dashboard Development
6. Business Insights Visualization

The final solution includes both backend data processing and a frontend analytics dashboard.

# How to Run the Project

## Backend

Navigate to the backend project:

```bash
cd backend/ecommerce-analytics-platform
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run data processing scripts as required.

## Frontend

Navigate to the frontend project:

```bash
cd frontend/ecommerce-analytics-web
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
